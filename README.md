## Используемые технологии
- Spring Boot 3.0
- Spring Security
- JSON Web Tokens (JWT)
- BCrypt
- Maven

## Функции
   - Регистрация и вход пользователей с помощью аутентификации JWT
   - Шифрование паролей с помощью BCrypt
   - Ролевая авторизация с помощью Spring Security
   - Механизм выхода из системы
   - Обновление токена
